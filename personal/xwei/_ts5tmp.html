MY WORKS

iOS Automation Framework


This automation framework was created to help build the automation process for iOS apps developer by SugarCRM.

The framework takes in existing test scripts created for web applications, and convert them into javascripts usable by instruments. This way, there's no need to implement completely new sets of test cases.

iOS Automation Utilizing Instruments

WebApp - B2G WebAPI


This web app was created to help smoke-testing the newly invented Firefox OS devices. It is used to monitor the status of each available WebAPI in Firefox OS, and generate a status report on the fly.

It can also be installed to the Firefox OS devices as an app, so that the devices can start with this app being loaded to the OS and perform tests onwards.

Test Page For Geolocation API

Mobile App - XE.COM


This is the Blackberry version of the widely used currency convertion app from xe.com.

I'm mainly responsible for maintaining the app according to bug reports and user feedbacks. Also, some new features were added to the app during my stay at xe.com as well.
